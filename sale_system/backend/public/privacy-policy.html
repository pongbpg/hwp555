นโยบายความเป็นส่วนตัว (Privacy Policy)

อัปเดตล่าสุด: มกราคม 2026

1. บทนำ

Sale System ("ระบบ", "เรา", "ของเรา") มีความมุ่งมั่นที่จะปกป้องความเป็นส่วนตัวของคุณ นโยบายนี้อธิบายว่าเราเก็บรวบรวม ใช้งาน และปกป้องข้อมูลส่วนบุคคลของคุณอย่างไร

2. ข้อมูลที่เรารวบรวม

เรารวบรวมข้อมูลจาก Facebook เพื่อให้บริการในระบบของเรา:

  • ข้อมูลหน้า (Page Information): ชื่อหน้า, ID หน้า, รูปโปรไฟล์, ชื่อผู้ดูแล
  • ข้อมูลข้อความ (Message Data): ข้อความจากลูกค้า, ประเภทข้อความ, เวลาส่ง
  • ข้อมูลลูกค้า (Customer Data): ชื่อลูกค้า, Facebook ID, URL โปรไฟล์
  • ข้อมูลการเข้าถึง (Access Token): Token สำหรับการเชื่อมต่อกับ Facebook

3. วิธีที่เราใช้ข้อมูลของคุณ

  • จัดการข้อความและการสนทนากับลูกค้า
  • ติดตามการสั่งซื้อและสถานะการจัดส่ง
  • วิเคราะห์ประสิทธิภาพของหน้าและแคมเปญ
  • ปรับปรุงบริการและประสบการณ์ผู้ใช้
  • ปฏิบัติตามกฎหมายและข้อกำหนดทางกฎหมาย

4. การแบ่งปันข้อมูล

เรา ไม่แบ่งปัน ข้อมูลส่วนบุคคลของคุณกับบุคคลที่สามยกเว้น:

  • Facebook Inc. (ผ่านการเชื่อมต่อ OAuth)
  • บริษัทสำเร็จการบ้านของเรา (ถ้ามี)
  • ตามที่กฎหมายกำหนด (ศาล, หน่วยงานรัฐ)

5. การเก็บรักษาข้อมูล

เราเก็บข้อมูลของคุณตราบเท่าที่จำเป็นเพื่อให้บริการ คุณสามารถร้องขอให้ลบข้อมูลของคุณได้ตลอดเวลา

6. ความปลอดภัยของข้อมูล

เรามีมาตรการความปลอดภัยที่เหมาะสม เช่น:

  • เข้ารหัส (Encryption) สำหรับข้อมูลที่ละเอียดอ่อน
  • การยืนยันตัวตน (Authentication) แบบ JWT token
  • การจำกัดการเข้าถึง (Access Control)
  • การตรวจสอบและอัปเดตระบบปกป้องรหัสผ่าน

7. สิทธิของคุณ

คุณมีสิทธิ:

  • เข้าถึงข้อมูลส่วนบุคคลของคุณ
  • ขอให้แก้ไขข้อมูลที่ไม่ถูกต้อง
  • ร้องขอให้ลบข้อมูล (Right to be Forgotten)
  • ถอนความยินยอม
  • ติดต่อเราเพื่อได้รับการช่วยเหลือ

8. การติดต่อ

หากคุณมีคำถามเกี่ยวกับนโยบายความเป็นส่วนตัวนี้ โปรดติดต่อ:

Sale System Support
Email: support@salessystem.local

9. การเปลี่ยนแปลงนโยบาย

เราขอสิทธิในการแก้ไขนโยบายนี้ได้ตลอดเวลา การเปลี่ยนแปลงจะมีผลบังคับใช้ทันทีที่เผยแพร่ในหน้านี้
